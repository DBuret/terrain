== Introduction 

This projet is aimed at discovering fragment shaders.

Easiest way to see the result is on https://www.shadertoy.com/view/MdjSRG[shadertoy].

TIP: All of this comes from reading http://www.iquilezles.org/[Inigo Quilez website] after discovering his https://www.shadertoy.com/view/MdX3Rr[evelated] fragment shader.

== concepts used

* fragment shader
* procedural terrain generation through perlin noise
* raymarching
